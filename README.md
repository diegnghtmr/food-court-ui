# Plazoleta UI

Sistema de gestión de plazoletas de comidas con arquitectura de microservicios.

## Descripción

Aplicación web desarrollada con Next.js para la gestión integral de plazoletas de comidas, incluyendo:

- Gestión de restaurantes y platos
- Sistema de pedidos
- Roles de usuario (Admin, Propietario, Empleado, Cliente)
- Trazabilidad de operaciones

## Tecnologías

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Radix UI

## Instalación

```bash
npm install
# o
pnpm install
```

## Desarrollo

```bash
npm run dev
# o
pnpm dev
```
